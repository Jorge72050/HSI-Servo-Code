
The code used to generate the rotation of the 360 servos.

# Files

- InputProcessing - The main file, includes contents of wifi connection and imports from RotateFunctions to generate movement.

- RotateFunctions - A header file that is imported into InputProcessing
   - Codes the rotation of the 360 servos with a delay implemented to ensure they rotate as long 
     as we want.

- PythonWifi - The attempt at using Python to establish a WiFi connection.
